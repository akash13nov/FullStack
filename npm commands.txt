BackEnd:
1. Add a gitignore file :-           touch .gitignore
2. Create a server :-               npm init -y
3. Create a server :-               npm i express
4. Server is started :-             npx nodemon server.js
5. Connect database to server :-    npm i mongoose
6. install .env package :-          npm i dotenv
7. cors package :-                  npm i cors





FrontEnd:
1. create react project :-          npm create vite@latest .
2. install axios (for API Call) :-  npm i axios
3. build frontend project :-        npm run build
